rotation-invariant:旋转不变，当图片旋转是，图片中的角度不会改变，还是可以检测到
Scale Invariant：尺寸不变，

当图片缩小或者扩大时，某些算法检测到的角跟scale之前检测到的结果会不一样，也就是说，对于某些算法，当对图片进行scale后，一些角不再是角
